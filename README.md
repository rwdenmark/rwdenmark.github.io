# rwdenmark.github.io

Personal portfolio site for Ryan Denmark. Live at <https://rwdenmark.github.io/>.

## Stack

Static HTML/CSS, no build step. Will likely move to Next.js or similar once project cards and content are added.

## Deploy

GitHub Pages serves `main` from the repo root automatically. Push to `main`, wait a minute, site updates.

## Local preview

Just open `index.html` in a browser. No tooling required at this stage.

## Files

- `index.html` — placeholder landing page
- `.nojekyll` — opt out of Jekyll processing (in case the site moves to a non-Jekyll generator)
- `.gitignore` — standard ignores
